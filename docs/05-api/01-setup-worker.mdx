---
title: setupWorker()
---

A function that configures the mocking Service Worker instance.

## Examples

```js showLineNumbers
import { setupWorker, rest } from 'msw'

const worker = setupWorker(
  // Provide request handlers
  rest.get('/user/:userId', (req, res, ctx) => {
    return res(
      ctx.json({
        firstName: 'John',
        lastName: 'Maverick',
      }),
    )
  }),
)

// Start the Mock Service Worker
worker.start()
```
