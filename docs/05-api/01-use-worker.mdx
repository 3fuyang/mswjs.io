---
title: useWorker()
---

A function that configures the mocking Service Worker instance.

## Examples

```js showLineNumbers
import { composeMocks, rest } from 'msw'

const { start } = composeMocks(
  // Provide request handlers
  rest.get('/user/:userId', (req, res, ctx) => {
    return res(
      ctx.json({
        firstName: 'John',
        lastName: 'Maverick',
      }),
    )
  }),
)

// Start the Mock Service Worker
start()
```
